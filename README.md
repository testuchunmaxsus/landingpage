# Fergana Soft — Landing Page

Raqamli yechimlar studiyasi uchun bir sahifali landing.

## Tarkib

- `index.html` — to'liq sahifa (HTML + CSS + JS bitta faylda)
- `assets/` — jamoa a'zolarining fotosuratlari

## Texnologiyalar

- HTML5 / CSS3 (CSS variables, Grid, Flexbox)
- Vanilla JavaScript (Canvas API, IntersectionObserver, localStorage)
- Google Fonts: Fraunces, Space Mono, JetBrains Mono

## Imkoniyatlar

- Dark va Light mavzu (toggle nav'da, localStorage'da saqlanadi)
- Custom kursor va smooth animatsiyalar
- Canvas particle fon
- Scroll-trigger animatsiyalar
- Responsive (1024px va 600px breakpoint'lar)
- Loyihalar marquee va loader

## Ishga tushirish

`index.html` faylni brauzerda ochish kifoya — build kerak emas.
